# 📊 Anime Analytics Project

## 📌 Overview

This is a personal data analysis project based on my anime watch history from MyAnimeList. The goal of this project is to explore viewing patterns, preferences, and trends using data analysis techniques.

---

## 📂 Data Source

The dataset was obtained from my MyAnimeList account export.
It contains information about watched anime, ratings, genres, and watch status.

---

## 🧹 Data Cleaning

Before analysis, the raw dataset was processed to ensure privacy and usability:

* Removed all personal and sensitive information
* Handled missing and inconsistent values
* Standardized column formats
* Filtered relevant fields for analysis

---

## 🎯 Objectives

* Analyze personal anime watching trends
* Identify favorite genres and themes
* Study rating patterns over time
* Build visual insights from the dataset

---

## 🛠️ Tools & Technologies

* Python libraries including Pandas, NumPy, Seaborn, and Matplotlib for data cleaning, analysis, and visualization.
* Power BI / Tableau (for visualization)

---
## 📁 Data Files

* **`myanimelist.xml`** – Raw data export from MyAnimeList
* **`myanimefile.csv`** – Processed and cleaned dataset used for analysis
* **`MyAnimeList.ipynb`** – Notebook documenting the full workflow (data cleaning, EDA, and visualization)
